jobExecutor.cpp : 
    περιεχει τη main μαζι με δυο κομματια το parent code και το child code.

trie.cpp: 
    περιεχει τη δομη του trie οπως εχει περιγραφει και στην εκφωνηση δηλαδη καθε κομβος εχει εναν char για το γραμμα και τρεις δεικτες
    δυο δεικτες σε κομβους trienode και εναν σε postinglist και ολες τις συναρτησεις σχετικες με το trie οπως: insertTrie, freeTrie καθως και συναρτησεις για τη posting list.

commands.cpp:
    περιεχει τις συναρτησεις που καλουνται μεσω των commands που δινονται απο το χρηστη (/search,/maxcount,/mincount,/wc,/exit) και αλλες βοηθητικες που χρησιμοποιουνται απο αυτες.

list.cpp: 
    περιεχει μια απλη δομη λιστας στην οποια αποθηκευονται οι γραμμες και το πληθος των φορων που εμφανιζεται η λεξη στην γραμμη (αποτελει μελος του postinglistnode).

searchlist.cpp:
    δομη λιστας που χρησιμοποιειται απο την /search για να αποθηκευει τα μοναδικα αρχεια που θα χρειαστει να επιστρεψει και να μην υπαρχουν διπλοτυπα.

intlist.cpp:
    δομη λιστας που αποτελει μελος του searchlistnode και αποθηκευει τις μοναδικες γραμμες ενος αρχειου που θα επιστραφουν απο τη search.

ΠΡΟΣΟΧΗ: Καθε γραμμη κειμενου δεν θα πρεπει να ξεπερναει σε μηκος το μεγεθος buffer του pipe.

Compile: make, make clean
Execute: ./jobExecutor -d docfile -w 10
