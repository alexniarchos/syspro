minisearch.cpp : 
    περιεχει τη main με ολες τις εντολες (/search,/df,/tf,/exit)

trie.cpp: 
    περιεχει τη δομη του trie οπως εχει περιγραφει και στην εκφωνηση δηλαδη καθε κομβος εχει εναν char για το γραμμα και τρεις δεικτες
    δυο δεικτες σε κομβους trienode και εναν σε postinglist, η οποια περιεχει dfcounter για να ξερουμε σε ποσα docs υπαρχει η λεξη αλλα και ποσες φορες
    περιεχει επισης και ολες τις συναρτησεις σχετικες με το trie οπως: insertTrie, freeTrie καθως και συναρτησεις για τη posting list

commands.cpp:
    περιεχει τις συναρτησεις που καλουνται μεσω των commands που δινονται απο το χρηστη (/search,/df,/tf) και αλλες βοηθητικες που χρησιμοποιουνται απο αυτες

maxheap.cpp: 
    περιεχει τις συναρτησεις για το maxheap το οποιο ειναι μια τροποποιημενη μορφη του minheap που ειχα φτιαξει για το Μαθημα των Λειτουργικων.

list.cpp: 
    περιεχει μια απλη δομη λιστας ετσι ωστε να αποθηκευω τα doc_ids για τα οποια υπολογιζω το score και τα βαζω στο maxheap

Compile: make, make clean
Execute: ./minisearch -i file -k 10
